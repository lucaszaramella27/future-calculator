# Calculadora Moderna (Electron)

Calculadora minimalista em tema escuro, com teclado, % no padrão de calculadora, e empacotamento para Windows via Electron + electron-builder.

## Como rodar (modo dev)

### Requisitos
- Node.js LTS instalado (recomendado)
- Windows/macOS/Linux (build .exe é no Windows)

### Comandos
```bash
# 1) entrar na pasta
cd calculadora-electron

# 2) instalar dependências
npm install

# 3) abrir o app
npm run dev
